## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository with user management and role based authorizations

## Installation

```bash
$ npm install
```

## Migrate database

```bash
npx prisma migrate dev
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# generate context once
$ npm run test:context

# unit tests
$ npm run mocha

# e2e tests
$ npm run mocha:e2e

# test coverage
$ npm run mocha:cov
```
